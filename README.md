# STUDENT LIFEOS

## Description
I made this app because i wanted a space where i could put all my school stuff together in one page. Yes, i know Notion is similar but I dont think notion can put a pomodoro timer, can it?
I know you can add it using embeds but that takes ages. So i just thought why not make an app with it already in it. I know its not all that great but its a first try using firebase 
for authentication with Google. I really like it. I liek that it has many settings like different themes/colors and different overall fonts. Its all my design but I did use some ai
for debugging and learning to use firebase and how to connect it to everything and have everything persist throughout. It was a really tough project since I had to learn a lot
while also debugging all the bugs. And I tell you, there were a ton of bugs. Every change I made resulted in a bug because I wanted everything to be stored by firebase using googleAuth
but it obviously couldnt be easy. But I hope you guys like it!
